Support

How to get help with Ultra Open Client.

Ultra Open Client is a free, open-source iOS app maintained on a volunteer basis. The fastest way to get help, report a bug, or ask a question is through the GitHub repository for this project.

Report an issue

Bugs, crashes, and feature requests are tracked publicly on GitHub. Open a new issue with a short description of what happened, your iPhone model, your iOS version, and steps to reproduce the problem if possible.

Frequently asked questions

Does the app work offline?

Yes. The default mode uses Apple Intelligence running directly on your iPhone, with no network access required. Responses are generated entirely on-device.

Which devices support Apple Intelligence mode?

Apple Intelligence requires an iPhone 15 Pro or newer, running iOS 26 or later. On older devices the app will fall back to other available backends.

Do I need an account to use the app?

No. Anonymous mode works out of the box without creating an account. You can also Sign in with Apple for private on-device chat synced across your devices via iCloud, or use phone sign-in for the full feature set.

What data is collected?

None by the developer. The app does not include analytics, crash reporters, or third-party trackers. See the Privacy Policy for full details.

How do I reset my local chat history?

Open the Settings sheet inside the app and choose "Clear chat history". On-device chats are stored locally via SwiftData and can be removed at any time.

Security reports

If you believe you have found a security vulnerability, please do not open a public issue. File it as a private security advisory on GitHub so it can be triaged responsibly.
